## Neovim

Instalação do gerenciador de plugins

```sh
curl -fLo ~/.config/nvim/autoload/plug.vim --create-dirs \
  https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim
```

Instalação dos plugins

```sh
:PlugInstall
```
